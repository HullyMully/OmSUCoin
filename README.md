# OmSUCoin - Система учета активности студентов на блокчейне

OmSUCoin - это децентрализованная система для учета и поощрения студенческой активности в ОмГУ с использованием блокчейн-технологий.

## Возможности системы

### Для студентов

1. **Регистрация и авторизация**
   - Вход через учетную запись ОмГУ
   - Создание профиля с указанием факультета и псевдонима
   - Привязка криптокошелька для получения токенов

2. **Управление профилем**
   - Изменение личной информации
   - Добавление/изменение адреса криптокошелька
   - Просмотр истории полученных токенов

3. **Участие в активностях**
   - Просмотр списка доступных активностей
   - Регистрация на активности
   - Отмена регистрации
   - Просмотр статуса заявок

4. **Просмотр статистики**
   - Таблица лидеров по количеству токенов
   - История личных транзакций
   - Доступные награды за токены

### Для администраторов

1. **Управление активностями**
   - Создание новых активностей с указанием:
     * Названия и описания
     * Даты и места проведения
     * Количества токенов за участие
     * Максимального числа участников
   - Редактирование существующих активностей
   - Просмотр списка участников

2. **Управление участниками**
   - Подтверждение заявок на участие
   - Отклонение заявок
   - Просмотр информации о студентах
   - Начисление токенов участникам

3. **Начисление токенов**
   - Выбор получателей (все/подтвержденные/выбранные)
   - Указание количества токенов
   - Добавление примечания к транзакции
   - Массовое начисление через смарт-контракт

## Пошаговые инструкции

### Для студентов

1. **Как привязать криптокошелек:**
   - Войдите в систему
   - Перейдите в раздел "Профиль"
   - Нажмите "Добавить кошелек"
   - Введите адрес кошелька BSC
   - Сохраните изменения

2. **Как зарегистрироваться на активность:**
   - Откройте раздел "Активности"
   - Выберите интересующую активность
   - Нажмите "Зарегистрироваться"
   - Ожидайте подтверждения от администратора

3. **Как посмотреть свои токены:**
   - Перейдите в раздел "Профиль"
   - Просмотрите баланс токенов
   - В разделе "История" доступны все транзакции

### Для администраторов

1. **Как создать активность:**
   - Перейдите в раздел "Активности"
   - Нажмите "Создать активность"
   - Заполните форму с деталями:
     * Название и описание
     * Дата и место проведения
     * Количество токенов
     * Лимит участников
   - Нажмите "Создать"

2. **Как управлять участниками:**
   - Откройте активность
   - Перейдите к списку участников
   - Используйте кнопки ✓/× для подтверждения/отклонения
   - Просматривайте детали участников через кнопку (i)

3. **Как начислить токены:**
   - Откройте список участников активности
   - Нажмите "Начислить токены"
   - Выберите получателей
   - Укажите количество токенов
   - Добавьте примечание
   - Подтвердите начисление

## Требования к системе

- Современный веб-браузер с поддержкой Web3
- Криптокошелек с поддержкой BSC (для получения токенов)
- Учетная запись ОмГУ

## Настройка окружения

1. Клонируйте репозиторий
2. Установите зависимости: `npm install`
3. Создайте файл `.env` с необходимыми переменными:
   ```
   DATABASE_URL="..."
   BSC_PROVIDER_URL="..."
   OMSUCOIN_CONTRACT_ADDRESS="..."
   ADMIN_PRIVATE_KEY="..."
   ```
4. Запустите сервер: `npm run dev`

## Технологии

- Frontend: React, TypeScript, Tailwind CSS
- Backend: Node.js, Express, PostgreSQL
- Blockchain: BSC, Web3.js, Solidity
